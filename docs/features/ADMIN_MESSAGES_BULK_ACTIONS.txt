✅ FITUR BULK ACTIONS DI ADMIN MESSAGES - SELESAI
================================================

FITUR YANG DITAMBAHKAN:
----------------------

1. ✅ Checkbox untuk select individual messages
   - Setiap message punya checkbox di sebelah kiri
   - Bisa klik untuk select/deselect message

2. ✅ Select All / Deselect All
   - Button "Select All" untuk tandai semua messages sekaligus
   - Otomatis berubah jadi "Deselect All" kalau semua sudah dipilih
   - Menampilkan jumlah messages yang dipilih (contoh: "5 selected")

3. ✅ Mark as Read (Bulk)
   - Button "Mark as Read" muncul kalau ada messages yang dipilih
   - Tandai semua selected messages sebagai "read" sekaligus
   - Menampilkan toast notification berapa messages yang di-mark

4. ✅ Delete Selected (Bulk)
   - Button "Delete Selected" muncul kalau ada messages yang dipilih
   - Hapus semua selected messages sekaligus
   - Ada confirmation dialog sebelum delete
   - Menampilkan toast notification berapa messages yang dihapus

CARA PAKAI:
-----------

1. Buka Admin Messages page
2. Klik checkbox di sebelah message untuk select individual message
3. Atau klik "Select All" untuk select semua messages
4. Setelah select, akan muncul 2 button:
   - "Mark as Read" - untuk tandai semua selected messages sebagai read
   - "Delete Selected" - untuk hapus semua selected messages
5. Klik button yang diinginkan
6. Untuk delete, akan muncul confirmation dialog
7. Selesai!
