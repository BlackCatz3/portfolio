üìã LANGKAH TEST:
=================================================================

1. Buka browser INCOGNITO/PRIVATE MODE (penting untuk clear cache!)
   - Chrome: Ctrl + Shift + N
   - Firefox: Ctrl + Shift + P

2. Login ke admin:
   https://4leafclover.id/admin/login
   
3. Masuk ke Home section:
   https://4leafclover.id/admin/home

4. Di bagian "Profile Image":
   - Klik "Choose File"
   - Pilih gambar BARU (jangan pakai yang sudah pernah di-upload!)
   - Klik "Save Media"

5. Tunggu sampai muncul notifikasi "Success"

6. Buka browser console (F12 ‚Üí Console tab)

7. Jalankan perintah ini di console:
   ```javascript
   fetch('https://api.4leafclover.id/api/about', {
     headers: {'Authorization': 'Bearer ' + localStorage.getItem('token')}
   })
   .then(r => r.json())
   .then(data => {
     console.log('Profile Image Path:', data.profile_image);
     console.log('Full URL:', 'https://api.4leafclover.id' + data.profile_image);
     
     // Test akses gambar
     fetch('https://api.4leafclover.id' + data.profile_image)
       .then(r => console.log('Image Status:', r.status, r.statusText))
       .catch(e => console.error('Image Error:', e));
   });
   ```

8. Lihat output di console:
   - Harusnya dapat: "Image Status: 200 OK"
   - Jika dapat 404: File tidak ada di server
   - Jika dapat CORS error: Nginx config belum benar

9. Refresh halaman (Ctrl + Shift + R untuk hard refresh)

10. Gambar seharusnya muncul!
