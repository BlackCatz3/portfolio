4. Submit ‚Üí Dapat 2 keys:
   - Site Key (untuk frontend)
   - Secret Key (untuk backend)

5. Update keys:
   
   **Frontend** (porto/.env.production):
   VITE_RECAPTCHA_SITE_KEY=your_real_site_key_here
   
   **Backend** (VPS environment variable):
   export RECAPTCHA_SECRET_KEY=your_real_secret_key_here

   environment:
     - RECAPTCHA_SECRET_KEY=your_real_secret_key_here

6. Rebuild frontend:
   npm run build
   
7. Restart backend:
   docker compose restart backend
