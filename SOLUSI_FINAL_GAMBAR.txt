SOLUSI 5: CEK CONSOLE BROWSER
------------------------------
1. Buka https://4leafclover.id/admin/home
2. Tekan F12
3. Klik tab "Console"
4. Refresh halaman (F5)
5. Lihat apakah ada error merah

Jika ada error seperti:
- "Mixed Content" → Berarti masih pakai localhost
- "CORS error" → Berarti Nginx CORS belum benar
- "404 Not Found" → Berarti file tidak ada
- "403 Forbidden" → Berarti permission salah


SOLUSI 6: CEK NETWORK REQUEST
------------------------------
1. Tekan F12
2. Klik tab "Network"
3. Refresh halaman (F5)
4. Cari request ke gambar (1767515119776-599079848.jpg)
5. Klik request tersebut
6. Lihat:
   - Request URL: Harus https://api.4leafclover.id/uploads/...
   - Status Code: Harus 200 OK
   - Response Headers: Harus ada Access-Control-Allow-Origin

Jika Request URL masih http://localhost:5000/uploads/...
→ Berarti frontend belum deploy versi terbaru
